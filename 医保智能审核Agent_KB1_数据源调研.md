# 医保智能审核 Agent · KB1 权威数据源调研（鹰眼项目）

> 调研时间：2026-06-18 ｜ 锚定范围：全国基线 + 江苏省覆盖层
> 核查方式：联网检索国家医保局（NHSA）及江苏省医保局官网，逐条验证 URL 现行有效性
> 重要约定：凡未亲自验证到下载入口的，一律标注「待核实」，不编造文号或链接

---

## 0. 三个必须先讲清楚的结论（影响整个抓取架构）

1. **「两库」全量 24.7 万条知识点没有"一键下载"。** 那是**出版物**（《医疗保障基金智能监管规则库、知识库（2025年版）》，已出版发行，扫码获取电子版需购书）。官网真正可抓的是**分批公开的规则/知识点公告**，挂在「通知公告」栏目（col109），**每批都带 .xlsx + .pdf 双附件**，截至本次检索已发布到**第十三批**（2026-06-01）。所以 KB1 的可程序化来源 = 逐批累积，而非一次拉全量。

2. **NHSA 与江苏省医保局都用同一套国产 CMS（中标/Chinaweb 系），页面是服务端渲染的静态 HTML，不是 Vue。** 这对抓取是好消息：列表页 `…/col/colXXX/index.html`、正文页 `…/art/YYYY/M/D/art_{colid}_{artid}.html`、附件直链 `…/module/download/downfile.jsp?classid=0&filename={hash}.{ext}`。可直接用 requests + 解析，无需 headless 浏览器。

3. **江苏省"2025版药品目录数据库"这个说法需要修正。** 江苏省在 2026 年之前执行的是**国家2024年版目录**，其省级数据库对外标注为「2024年版」，按维护批次以 `（年版）（YYYYMMDD）` 形式滚动公示（如 20250225、20250427）。国家2025版目录 2026-01-01 才生效，因此江苏对应的"2025版"省级数据库属于 **2026 年的发布物**，需在江苏省局通知公告里持续蹲守，目前**待核实**是否已挂网。

## 3. 权威性判定标准（执行口径）

**为何算权威**：①发布主体为国家医保局/省医保局/国务院；②有正式文号（局令、医保发〔年〕号、医保办函〔年〕号）或挂在官网法定主动公开栏目；③现行有效（核对施行日期、是否被新版替代）。

**如何交叉验证**：
- 国家级条例/细则 → 中国政府网 + 国务院公报 + ≥1个省局转载三处比对文号与施行日期。
- 编码数据 → code.nhsa.gov.cn 贯标平台 + 附带MD5校验。
- 省级文件 → 省局原文 + 地市转发版（地市转发常带完整PDF附件，可互证）。
- 文号检索 → 直接以"苏医保发〔2025〕20号"等文号在官网站内/政府网搜索定位原文。

**红旗（降级为"参考"或丢弃）**：
- 非政府域名转载（搜狐/新浪/医药魔方/CSDN/知识星球/律所解读）——可用于发现线索，**不得作为入库主源**。
- 无文号的截图、PDF来源不明、过期版本未标注施行/失效日期。
- 第三方"整理站"（如 tao-shu.com、huimei.com 等）——仅作发现入口，原文必须回到 .gov.cn。

**禁止作为权威主源**：CSDN、知识星球、律所/咨询解读代替原文、自媒体二次整理。这些只能进 `source_tier=reference`。

## 4. 抓取可行性评估（爬虫设计输入）

**全站共性（NHSA + 江苏，中标CMS）**：
- 页面技术：**静态HTML，服务端渲染**，无需headless（requests+lxml即可）。
- URL规律：列表 `…/col/col{colid}/index.html`（翻页参数）；正文 `…/art/YYYY/M/D/art_{colid}_{artid}.html`；附件 `…/module/download/downfile.jsp?classid=0&filename={hash}.{ext}`。
- artid 近似递增，但**不要靠纯遍历id**（会漏栏目、踩软404）；以列表页驱动更稳。

## 7. Gemini 补充整合（2026-06-18）

### 7.1 规则解构：88 类 ≠ 88 条独立逻辑

国家局「88 类、24.7 万条」在工程上多为：

**规则模板（几十个）× 医保目录编码（数万）= 实例化知识点**

爬虫入库时应保留 `rule_category`（如「药品限适应症」）与 `knowledge_point` 明细，而非当作 24 万个独立 checker。

### 7.2 栏目索引勘误（Gemini vs Claude）

| Gemini 写法 | 正确入口（Claude 核验） | 说明 |
|---|---|---|
| col106 作两库主入口 | **col109** 通知公告 | 分批 xlsx 在 col109，col106 为基金监管动态 |
| col104 搜药品目录 | art_104_18970 等正文页 | 列表翻页 + 正文附件 |
| Playwright 默认必需 | **fetch + cheerio 优先** | 静态 CMS；403 时再降级 Playwright |

### 7.3 权威性白名单（入库清洗）

- 域名：仅 `*.gov.cn` 作主源
- 文号：正则归一 `国医保发〔202X〕X号`（统一六角括号 `〔〕`，剔除空格）
- 施行日期：正文末尾 `(自\d{4}年\d{1,2}月\d{1,2}日起.*?施行)` 优先于发布日期
- 废止：定期抓「失效和废止规范性文件目录」，对应条目 `effective_to` 置值
- 红旗：自媒体、律所解读、无文号截图 → `source_tier=reference`，不可作 audit 依据
