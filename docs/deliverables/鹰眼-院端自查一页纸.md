## 一、您正在面对的现实

- **强制自查常态化**：每年 12 个重点领域（肿瘤、骨科、麻醉、重症、心血管…）全国铺开，要扫的是**过去两年、多科室**的历史案卷。
- **飞检核查的是"自查质量"，不是"报完就结束"**：四不两直进场常带大数据疑点；自查走过场 → 从重处理、曝光、支付资格记分；**主动退回 → 从轻**。
- **痛在哪**：①历史案卷海量，人工逐份翻不完；②语义类问题（靶向药指征、分解住院、费用与病程不符）字段规则扫不出；③"查了改了"拿不出**可核验的台账**。

> 一句话：**国家两库帮您少犯新错；鹰眼帮您把旧账查清、查清能写进报告、改完能证明。**

## 二、鹰眼怎么帮（体检模式 · 嵌入日常病案质控）

| 环节 | 院端痛点 | 鹰眼能力 |
|---|---|---|
| **查** | 12 领域×两年，翻不完 | **优先队列**按风险指数排序，先查高风险案卷；批量稽核 |
| **查·语义** | 靶向药指征、分解住院、病程与收费不符 | **三方语义交叉**（费用—诊断—医嘱），出**三要素证据链**：证据定位+政策条款原文+推理 |
| **纠** | 不知改哪、科室口径不一 | **体检模式口径**：风险点 + 整改建议 + 主动退回清单 |
| **证** | 怕"说了改了没证据" | **导出自查清单 / 举证包 / 违规统计**，直接进自查台账 |
| **复检** | 改完不知还有没有漏 | 同案卷**复跑**：飞检前模拟、干净件零误报、申辩材料现成 |

**关键差异**：不是事后告警一摞，而是**每条都能点开看原文、附条款引用、经得起对质**。多 Agent"控辩裁"先把医院可预见的申诉理由过滤一遍——进入人工的，都是"已扛过一轮申诉"的。

## 三、为什么可信（敢被检验，也认边界）

- **引不出原文的疑点，不输出**（三要素门禁）；条款只引官方政策原文，带文号/生效日/核验状态——可点开「合规地基」面板逐条溯源。
- **取证可信度三件套**：合议去重（一笔钱算一次·防过罚）+ 覆盖度声明（答"查全了吗"）+ 置信度传播（OCR 低置信降权·反幻觉）。
- **诚实边界**：以高精确率（低误报）为先、避免误伤合规诊疗；**最终裁定由院方/稽核员做，产品不做 AI 自动处罚**；漏判兜底交全量筛查与人工复核。
- **数据合规**：病历本地处理、可私有化/离线、送 LLM 前 PII 脱敏。

## 四、交付物（可直接进自查台账）

- 《疑点核查清单》（含费用行、金额、条款依据、整改建议）
- 机构自查体检报告（高风险规则/科室画像、抽样指导）Markdown / 可打印 PDF
- 违规统计表、举证包；整改前后复跑对比

## 五、价值与试用

- **ROI 自证**：一次飞检拒付/罚款远超工具费；主动退回又可从轻——工具成本是其零头。
- **定价锚**：按**机构 / 科室年度订阅**；轻服务切入、不动 HIS、低集成。
- **冷启动**：不依赖大规模真实病历训练，规则驱动 + 专家标注少量金标，合规友好。
- **试用方式**：选 1–2 个重点科室、给一批历史案卷，**先跑一轮自查体检**看产出——能正确"不报"、每条疑点可对质，再谈范围。
